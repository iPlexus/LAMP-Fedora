# LAMP - Fedora
A script that allows you to run a LAMP on Fedora

## Planning list
- Add config for PHP XDebug in php.ini

## Installation
Follow the instructions below
```
git clone https://github.com/iPlexus/LAMP-Fedora
cd LAMP-Fedora
chmod +x fedora_lamp.sh
sudo ./fedora_lamp.sh
```
And that's it !
